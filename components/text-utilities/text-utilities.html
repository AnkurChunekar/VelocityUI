Text Utilities

Text Utilities are useful for providing different styles to text content.

Text Styles

  • Use "gray-text" class for gray text color.
  • Use "strike-text" class for strike text from center.
  • Use "center-align-text" class to align text content center.
  • Use "right-align-text" class to align text content right.
  • Use "left-align-text" class to align text content left.
  • Use "highlight-text" class for highlight text.
  • Use "light-weight-text" class for small font weight text.
  • Use "b" tag for bold text.
  • Use "primary-color-text" class for primary text color.
  • Use "secondary-color-text" class for secondary text color.
  • Use "small" tag for bold text.
