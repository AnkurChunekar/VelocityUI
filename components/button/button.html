Buttons

Buttons are one of the most useful components in any Web Applications. Velocity provides different range of buttons and their states.

Solid Buttons

Create a solid button using "btn" class.

Additionally you can add "btn-primary", "btn-secondary", "btn-dark", "btn-danger", "btn-success", and "btn-warn" classes to get desired button.g

Outline Buttons

To create Outline simply add "btn-outline" class.

Icon and Link buttons

Create link buttons using "btn-link" class.

To create an Icon button you can use "btn-w-icon" class and add your desired icon inside the parent button tag and include additional span tag for text content.

Use "disabled" class to create a disabled button.

Floating Icon buttons

"btn-floating" class, can be used to create floating button.

Different sizes of Buttons

Three different sizes are available for buttons. Use "btn-s" , "btn-md" , and "btn-lg" classes to get small, medium and large sized buttons respectively.
