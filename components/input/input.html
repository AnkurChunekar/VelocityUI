Input

Form Inputs are very useful for users to input some login/signup data, and they also provide validation.

Standard Input

Create standard input providing a "input-wrapper" class to the parent container of input tag.

Validation with input

To provide validation to the user add "validation" class along with "input-wrapper" class.

Radio inputs

To create radio inputs use "radio" class along with "input-wrapper" class.

Provide "disabled" class to disable the input.

Input with an Icon

To create searchbar input use "input-icon" class along with "input-wrapper" class.

Checkbox inputs

To create checkbox inputs use "checkbox" class, only change the type attribute of the input tag to "checkbox".

Provide "disabled" class to disable the input.
