Badge

A badge can display the number of notifications or the status of being online or offline, or can even be used it as a tag.

Avatar with a badge

Add Badges to the avatars using "badge-container" class, after "avatar" class.

Icons with Badges

Add Badges to the icons using "icon-container" class, along with "badge-container" class. Include your desired icon with a span tag.

Badges used as tags

Tag Badges can be created using "tag" class.

Additionally use "tag-primary", "tag-secondary", "tag-success", "tag-danger", "tag-danger", "tag-dark" and "tag-warn" to create desired tag.

"tag-round-border" class can be used give small border radius.
