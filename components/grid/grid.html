Grid

CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.

Note: Resize window's width to check responsiveness.

Two Column Grid

Both Columns equally resize with respect to the available space. Classes: "grid" and "col2-fluid".

Two Column Right Fluid.

Only 2nd Column will fit according to the available space. Classes: "grid" and "col2-fluid-r".

Auto Fit Grid.

Items inside Auto-fit Grid will automatically fit automatically inside parent container. They will have a minimum width of 300px. Classes: "grid" and "auto-fit".
