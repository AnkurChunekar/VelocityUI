Alerts

An alert is used to communicate an important message in a way that attracts the user's attention without interrupting the user.

Standard Alert

Create standard alert using the "alert" class.

Additionally use classes such as "alert-success", "alert-error", "alert-info", "alert-warn" to create respective components.

Well Done! you have successfully read an alert message.
Error! Something went wrong, Try again.
When in doubt, use this one to communicate with the user.
Warning! a simple warning, check out.

Dismissible Alert

Create dismissible alert using the "alert-dismissible-icon" class given to your desired dismiss icon.
