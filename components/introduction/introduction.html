Velocity UI

Hello and welcome!

Quickly design beautiful, responsive web pages with the help Velocity UI, without a lot of hassle using only HTML and CSS.

Installation

In order to use the components in your project, Copy and paste the code in the head tag of your html document, above all other style sheets.

Once you are done with the first step, you can easily use the components by just adding class names to your html elements.

Customisable

Components can be easily customised. In order to override the base classes, users can add their own class to the pre-existing classes, and style them seperately.
