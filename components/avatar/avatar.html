Avatar

An avatar is an image used as a user's profile picture, usually in a circular shape. It is displayed on the user's profile information page, to give them a distinctive visual identity.

Standard Avatar

Create standard avatar using the "avatar" class.

Different Sizes

Additionally use classes such as "avatar-lg", "avatar-md", "avatar-s", "avatar-xs" to create large, medium, small and x-small avatars respectively.

Text avatars

Text avatars can be used as fallbacks. Use "avatar-text" class to create a text avatar.
